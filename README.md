# LocalRp

![My Image](https://github.com/MrghtChannel/LocalRp/blob/main/1.png)

Цей проект включає сайт, лаунчер та адмін-панель для мультиплеєра FiveM. Нижче наведено поточний статус кожного компонента.

| Компонент        | Версія | Опис                                                                                       |
|------------------|--------|--------------------------------------------------------------------------------------------|
| Сайт (Website)   | 0.1    | Базова версія сайту.                                                                       |
| Лаунчер (Launcher) | 0.1    | Лаунчер для гри без системи оновлення та підключення (функціонал незабаром з'явиться).     |
| Адмін-панель     | -      | **Coming Soon** – Адмін-панель знаходиться в розробці і буде доступна незабаром.          |
| Радіо (Radio)    | -      | **Coming Soon** – Система радіо наразі знаходиться в розробці.                             |
